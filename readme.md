# NAGOMI PowerHAL — 吃到飽回本神器

專為 NAGOMI 自助餐設計的 AI 回本計算器。拍照自動辨識餐點，即時追蹤你的 ROI！

## 功能特色

**拍照辨識** — 對著盤子拍一張，AI 自動辨識品項與數量

**即時儀表板** — 追蹤消費金額、熱量攝取、回本進度

**智慧菜單** — CP 值排行榜，教你怎麼吃最划算

**雙模式計價** — 切換「市價」或「飯店定價」，自由選擇計算方式

## 使用方式

1. 打開網頁
2. 點擊「INPUT」拍照或手動輸入
3. 確認品項後按「COMMIT」
4. 在「SYSFS」查看即時進度

## 小技巧

- 午餐時段 (11:30-15:00) 自動套用午餐價格
- 晚餐時段 (17:30-21:30) 自動套用晚餐價格
- 點右上角書本圖示查看完整菜單與 CP 值排行
